A **headless CMS** is a content management system that provides content via APIs, leaving the frontend presentation up to you. Here, we’ll:

- **Simulate a headless CMS** with a Node.js server to serve story content.
- Build an **HTML-based interactive frontend** that fetches this content and lets users navigate through the story by making choices.

---

**Step 2: Set Up the Backend (Simulated Headless CMS)**

We’ll use **Node.js** and **Express** to create a server that acts as our headless CMS, delivering JSON data via API endpoints.

**2.1 Prerequisites**

- Ensure **Node.js** is installed (download from nodejs.org if needed).
- Open a terminal and create a project directory:bash
    
    ```bash
    mkdir interactive-story
    cd interactive-story
    ```
    

**2.2 Initialize the Project**

- Set up a new Node.js project and install Express:bash
    
    ```bash
    npm init -y
    npm install express
    ```
    

**2.3 Create the Server**

- Create a file named server.js and add this code:javascript
    
    ```jsx
    const express = require('express');
    const app = express();
    const port = 3000;
    
    // Sample story data
    const stories = [
      {
        id: 1,
        title: "The Mysterious Quest",
        sections: [
          {
            id: 1,
            text: "You wake up in a forest with two paths ahead. Do you take the shadowy trail or the sunny path?",
            choices: [
              { text: "Shadowy trail", nextSection: 2 },
              { text: "Sunny path", nextSection: 3 }
            ]
          },
          {
            id: 2,
            text: "The shadowy trail leads to a hidden cave with treasure! The end.",
            choices: []
          },
          {
            id: 3,
            text: "The sunny path brings you to a friendly village. The end.",
            choices: []
          }
        ]
      }
    ];
    
    // Get story title
    app.get('/api/stories/:id', (req, res) => {
      const storyId = parseInt(req.params.id);
      const story = stories.find(s => s.id === storyId);
      if (story) {
        res.json({ id: story.id, title: story.title });
      } else {
        res.status(404).send('Story not found');
      }
    });
    
    // Get a specific section
    app.get('/api/stories/:storyId/sections/:sectionId', (req, res) => {
      const storyId = parseInt(req.params.storyId);
      const sectionId = parseInt(req.params.sectionId);
      const story = stories.find(s => s.id === storyId);
      if (story) {
        const section = story.sections.find(sec => sec.id === sectionId);
        if (section) {
          res.json(section);
        } else {
          res.status(404).send('Section not found');
        }
      } else {
        res.status(404).send('Story not found');
      }
    });
    
    app.listen(port, () => {
      console.log(`Server running at http://localhost:${port}`);
    });
    ```
    

**2.4 Run the Server**

- Start the server:bash
    
    ```bash
    node server.js
    ```
    
- It’ll run on http://localhost:3000, serving story data via APIs.

**3.3 Add JavaScript for Interactivity**

- Create script.js:javascript
    
    ```jsx
    async function loadSection(storyId, sectionId) {
      const response = await fetch(`/api/stories/${storyId}/sections/${sectionId}`);
      const section = await response.json();
      const sectionText = document.getElementById('section-text');
      const choicesDiv = document.getElementById('choices');
    
      // Fade out, update, then fade in
      sectionText.classList.remove('loaded');
      setTimeout(() => {
        sectionText.innerText = section.text;
        sectionText.classList.add('loaded');
      }, 50);
    
      choicesDiv.innerHTML = '';
      if (section.choices.length === 0) {
        choicesDiv.innerHTML = '<p>The end. Refresh to restart!</p>';
      } else {
        section.choices.forEach(choice => {
          const button = document.createElement('button');
          button.innerText = choice.text;
          button.addEventListener('click', () => loadSection(storyId, choice.nextSection));
          choicesDiv.appendChild(button);
        });
      }
    }
    
    async function startStory(storyId) {
      const response = await fetch(`/api/stories/${storyId}`);
      const story = await response.json();
      document.getElementById('story-title').innerText = story.title;
      loadSection(storyId, 1); // Start with section 1
    }
    
    // Begin with story ID 1
    startStory(1);
    ```

        ```bash
        live-server
        ```
        
    - Open your browser to http://127.0.0.1:8080 (or the port live-server provides).
3. **Interact**:
    - Read the story, click choices, and watch the content update with smooth transitions!

- **Headless CMS**: The Node.js server acts as a content backend, separate from the frontend, mimicking a real headless CMS.
- **Interactive**: Users shape the story with their choices, making it engaging.
- **Dynamic**: Content loads on-demand via APIs with animated transitions.
- **Simple Yet Powerful**: Built with basic web technologies (HTML, CSS, JS), yet extensible.
