Template Details

Overview

This template provides a complete video chat solution with ASL interpreter embedding.

Components Included

  • Twilio/Zoom video integration code
  • Interpreter embedding widgets
  • Screen sharing functionality
  • Recording and playback features
  • UI components with accessibility features

Configuration

{
  "video_provider": "twilio",
  "api_key": "YOUR_API_KEY",
  "interpreter_service": "vsl_labs",
  "features": {
    "screen_sharing": true,
    "recording": true,
    "asl_embedding": true
  }
}

Overview

Embed ASL interpreters directly into your application interface.

Example Code

<!-- VSL Labs Interpreter Embedding -->
<div id="vsl-interpreter" class="interpreter-container">
  <iframe 
    src="https://vsl.labs/embed/{session_id}"
    width="400"
    height="300"
    frameborder="0"
    allow="camera; microphone"
  ></iframe>
</div>

<!-- PinkSync Full Suite -->
<script src="https://pinksync.com/widget.js"></script>
<script>
  PinkSync.init({
    apiKey: 'YOUR_API_KEY',
    services: ['interpreter', 'gloss', 'captions']
  });
</script>

Overview

Convert heavy financial context into accessible gloss ASL format.

API Usage

POST /api/asl/convert-gloss
Content-Type: application/json

{
  "text": "The compound interest on your investment...",
  "domain": "financial",
  "output_format": "video"
}

Response:
{
  "gloss_text": "INTEREST COMPOUND YOUR INVESTMENT...",
  "video_url": "https://pinksync.com/gloss/video/abc123",
  "confidence": 0.95
}

Overview

Complete framework for integrating with enterprise APIs.

Plugin Registration

from services.enterprise.plugin_registry import enterprise_registry, PluginType
from services.enterprise.api_plugins import BloombergAPIPlugin

# Register Bloomberg plugin
enterprise_registry.register_plugin(
    PluginType.API_CONNECTOR,
    'bloomberg',
    BloombergAPIPlugin,
    config={
        'api_key': 'YOUR_API_KEY',
        'base_url': 'https://api.bloomberg.com'
    }
)

# Execute plugin
result = enterprise_registry.execute_plugin(
    PluginType.API_CONNECTOR,
    'bloomberg',
    action='get_financial_data',
    symbols=['AAPL', 'GOOGL']
)
